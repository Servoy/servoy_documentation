Select a servoy.properties file that should be used in this WAR.
A servoy.properties file hold the server definitions (which servers the application server will connect to) and many other configuration options for servoy or the plugins.
You can also select here your own web.xml file if you want to add stuff to the generated WAR (like servlet filters or custom configuration)
Please take first a web.xml from an exporter WAR and extract this web.xml file out of it for modification.
