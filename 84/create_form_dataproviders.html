Add data providers to form

Select data providers to appear on the form. These data providers can be columns, calculations, form variables, global variables, and data providers by relation.

Simple configuration

If simple configuration enabled, user can choose from datasource columns, calculations or aggregates.Then the right table will show selected columns and the component type that will use the data provider.

Advanced configuration

In advanced configuration user can change default component types for String, Integer, Number, Date or Media data types, set spacing between fields or default size of components.

Also user can set label to fields, change label default component, spacing, size or set on top of component
