Form components are very comparable to regular forms. This is why all properties in the wizards are the same as the wizard of regular forms. However, there a few slight differences which are explained below.

A form component can be created by doing a right-click on the "Form Components" node inside your solution. You can also do a left-click on that node and click the icon in the detail panel of the Solution Explorer.

Extends

Inheritance can be applied here too. But here only based on other existing form components.

Template

As with forms, you can use existing templates here as well. However, you are only able to select existing form components instead of forms.
