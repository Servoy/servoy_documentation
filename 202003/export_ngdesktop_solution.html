This wizard creates a NgDesktop installer

Since version 2020.12 the autoupdate feature is supported. In order to use this feature an URL for future update files must be provided when creating the installer.

A NgDesktop version after 2020.12 will generated 3 files: the installer binary, the update binary and an yml file containing update details. For the customers that already installed the NgDesktop application, you must place the update binary and the yml file into the URL location specified in a previous export, this should stay the same between exports

NgDesktop will detect the update, download it in the background and display a desktop notification. The update will be installed on the next restart.
